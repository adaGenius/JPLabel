# JPLabel
匹配一段文字中的用户名/话题/链接/协议等, 并且为关键字着色的UILabel子类，还支持自定义高亮显示字符串功能。


## 具体使用以及介绍，请参见我的简书文章[《[iOS]0行代码集成UILabel字符串匹配》](http://www.jianshu.com/p/c20b8b44adb3)
